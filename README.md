# Devcontainer Template

## Using this template

### New repository
1. Click “Use this template” → Create a new repo.
2. The `.devcontainer/` folder will be included automatically.

### Existing repository
1. Clone this repo somewhere temporary.
2. Copy the `.devcontainer/` folder into your project:

```bash
cp -r devcontainer-template/.devcontainer /path/to/your/project/
